
=============================

Personal web site
-----------------------------------

* 🌍  This is my personal website. Im creating this as a connector between my social media accounts as well as another public copy of my projects outside of github.
I built the webpage from scratch, only using the basic html boilerplate provided by the emmet plugin. I pulled up several web dev personal websites for inspiration. 
I mirrored some aspects such as the rainbow background gradient and the button styles from other projects i saw and molded it to fit into my schema. 
The website was designed to be simple and for that reason I only use barebones html/css and vanilla javascript. I also used media quieries and webkits within the CSS to 
ensure mobile-responsiveness as well as to make the styles more dynamic. One thing I learned from this project is to design from a mobile viewport first. I didn't take 
that approach and spent a lot of time working backwords to ensure mobile responsiveness that could've been saved had I started from that perspective first. The project is still a work in progress.
